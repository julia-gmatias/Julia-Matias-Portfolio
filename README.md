# 🕹️ Portfólio Pessoal

Este é o meu portfólio pessoal desenvolvido com uma estética retro/pixel art. O projeto foi construído para apresentar minhas habilidades técnicas, formação acadêmica e projetos realizados no ecossistema de desenvolvimento Back-end.

## 🚀 Tecnologias Utilizadas

*   **HTML5** - Estruturação semântica.
*   **CSS3** - Estilização moderna com variáveis (Custom Properties) e Flexbox.
*   **JavaScript** - Lógica para alternância de tema (Dark/Light mode).
*   **Google Fonts** - Fonte 'Press Start 2P' para a estética gamer.

## ✨ Funcionalidades

*   **Design Responsivo:** Adaptado para dispositivos móveis e desktops.
*   **Navegação Suave:** Menu fixo com scroll suave entre as seções.
*   **Formulário de Contato:** Estrutura pronta para integração de mensagens.

## 📁 Estrutura do Projeto

*   `index.html`: Estrutura principal do site.
*   `style.css`: Toda a identidade visual e variáveis de cores.
*   `script.js`: Lógica de persistência do tema e interatividade.

## 👩‍💻 Sobre Mim

Sou desenvolvedora backend Java em formação, estudante de Ciência da Computação e Programadora de Sistemas certificada. Atualmente foco meus estudos em:
*   Java e Spring Boot
*   Infraestrutura Cloud (AWS)
*   Banco de Dados SQL
